# Axionax Deploy

Deployment configurations and infrastructure for Axionax nodes.

## Quick Start

\\\ash
docker-compose up
\\\

## Contents

- **Dockerfile** - Docker image for Axionax nodes
- **environments/** - Configuration files for different environments
- **scripts/** - Setup and deployment scripts

## Documentation

See [docs.axionax.org/deployment](https://docs.axionax.org) for complete documentation.

- [axionax-core](https://github.com/axionaxprotocol/axionax-core) - Protocol core
- [axionax-sdk-ts](https://github.com/axionaxprotocol/axionax-sdk-ts) - TypeScript SDK
- [axionax-docs](https://github.com/axionaxprotocol/axionax-docs) - Documentation
